# Habit Tracker GAS

Google Sheets + Google Apps Script で作る習慣トラッカー。

## 機能

- **日次トラッキング**: 運動（種類・時間）、飲酒（本数）、体重を記録
- **ストリーク計算**: 運動連続日数・禁酒連続日数を自動計算
- **ダッシュボード**: 今週・今月の統計を一覧表示
- **リマインダー**: 毎晩21時に未記録の場合メールで通知
- **週次サマリー**: 毎週月曜朝にメールで週間レポート送信

## セットアップ

### 1. スプレッドシートを作成

Google Sheets で新しいスプレッドシートを作成。

### 2. GASプロジェクトを紐付け

```bash
# Script IDを.clasp.jsonに設定後
clasp push
```

### 3. 初期化

スプレッドシートを開き、メニュー「Habit Tracker」→「Initialize Sheet」を実行。

### 4. トリガー設定

メニュー「Habit Tracker」→「Setup Triggers」を実行。

| Date | Exercise Type | Duration (min) | Beers | Weight (kg) | Notes |
|------|--------------|-----------------|-------|-------------|-------|
| 2026-02-15 | Gym | 60 | 0 | 70.5 | Chest day |

### Dashboard シート

ストリーク、今週の統計、今月の統計を自動集計。

## トリガー

| 時間 | 機能 |
|------|------|
| 毎日 0:00 | 新しい行を自動追加 |
| 毎日 21:00 | 未記録リマインダーメール |
| 毎週月曜 8:00 | 週次サマリーメール |
